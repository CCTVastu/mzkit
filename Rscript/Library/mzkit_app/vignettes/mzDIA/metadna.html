annotationSet

create the kegg compound ms1 annotation query engine.

let annotationSet as function(kegg as any, 
precursors as any = "[M]+|[M+H]+|[M+H-H2O]+", 
mzdiff as any = "ppm:20", 
[<Environment>]) -> <generic> KEGGHandler {
    #
    # .NET API information
    #
    # module: mzkit.metaDNAInfer
    # LibPath: D:/mzkit/Rscript/Library/mzkit_app/assembly/net6.0
    # library: mzDIA.dll
    # package: "metadna"
    #
    return call R#.interop_[metaDNAInfer::CreateKEGGSearch](...);
}

Arguments

  • kegg: a set of kegg compound data
  • precursors: a character vector of the ms1 precursor ion names or
    a list of the given mzcalculator object models.
  • mzdiff: the mass tolerance value to match between the
    experiment m/z value and the reference m/z value
    which is calculated from the compound exact mass
    with a given specific ion precursor type.
  • env: -

Value

a data query engine model to run ms1 data search
for the kegg metaolite compounds.
